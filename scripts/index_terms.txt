# Back-of-book index terms for Programming Generative AI.
#
# One term per line:   pattern :: Key [:: slugs=a,b,c] [:: cs]
#
#   pattern  regex fragment, matched at word boundaries, case-insensitive
#            unless "cs" is given
#   Key      the index entry as it should appear in the book (defaults to the
#            pattern text when omitted; use ! for makeindex subentries)
#   slugs=   restrict the term to these chapter slugs (default: every chapter)
#   cs       case-sensitive match (proper nouns that double as common words)
#
# build_pdf.py injects \index{Key} after the FIRST match per chapter, in prose
# only (never inside verbatim blocks or LaTeX commands). A term that never
# matches simply does not appear in the index, so speculative entries are safe.
